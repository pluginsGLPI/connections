## Connections plugin for GLPI

[![License](https://img.shields.io/badge/License-GNU%20GPL%20v3-blue.svg?style=flat-square)](https://github.com/pluginsGLPI/connections/blob/master/LICENSE)
[![Translate](https://img.shields.io/badge/Translate-Transifex-cyan)](https://explore.transifex.com/teclib/glpi-plugin-connections/)


Français
========

Ce plugin vous permet de gérer les liaisons de votre réseau et de les associer à des éléments de l'inventaire.


English
=======

This plugin allows you to manage the connections of your network and associate them with elements of the inventory.
